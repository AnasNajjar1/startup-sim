# Startup Simulation

A startup management simulation built with:

- Next.js 14 (App Router)
- Supabase Auth
- PostgreSQL
- TailwindCSS
- Recharts

Users manage a startup by adjusting price, hiring engineers and sales staff, and controlling salary levels.

Each quarter runs a simulation calculating revenue, payroll, profit, and company survival.

Game ends when cash reaches zero or after 12 quarters.
